# Js video player

Android video player app (Kotlin + ExoPlayer) "Js video player".

This repository contains an Android Studio project that supports local file playback, streaming URLs, subtitles, and basic playback controls.

Features:
- ExoPlayer-based playback (local and streaming)
- File picker (Storage Access Framework)
- Play/pause/seek, volume controls, gestures
- Subtitles (local and URL)
- Recent files list
